Features.

Admin dashboard The admin dashboard requires authentication and allows you to:
  • Create one or more companies. Each company can have any number of restaurant menus.
  • Each menu can have its own theme, allowing you to customize the look and feel (pre-made themes).
  • Each time you create a new restaurant menu, a new URL and QR code are generated for it, so customers can access the menu.
  • Each menu is subdivided into sections, and each section contains menu items. Each menu item can have its own photo, description, and price tag.
Menu for restaurant customers
  • Allows any user (restaurant clients) to view the current menu.
  • Some themes also allow for additional functionality (e.g., WhatsApp button).

Technical details.

Menu Manager consists of two projects:

  • A backend in NodeJS with a PostgreSQL database managed trough sequelize
  • A frontend in NextJs that serves both the admin dashboard (client side render) and the menu itself (server side render)

Both are dockerized and can be run with docker-compose.

  • Login is managed trough AWS cognito, new user data is received troug AWS lambda functions
