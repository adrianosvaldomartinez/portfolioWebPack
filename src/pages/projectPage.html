Features

  1. Admin dashboard
    The admin dashboard requires authentication and allows you to:
    • Create one or more restaurant menus or edit the settings of an existing restaurant menu
    • Add sections to your restaurant menu (with optional headers and footers)
    • Add plates to your sections
    • Pick a theme for your menu
  2. Menu for restaurant customers
    Allows any user (restaurant clients) to see the actual menu.

Technical details

Menu Manager consists of two projects:

  • A backend in NodeJS with a PostgreSQL database
  • A frontend in NextJs that serves both the admin dashboard and the menu itself

Both are dockerized and can be run with docker-compose.
